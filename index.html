Open Type

A public package CDN for open typefaces that have no other place on the web to call home

How to use

Simply grab the CSS import snippet from your desired typeface below.

Cantarell
@import url('https://opentype.netlify.com/cantarell/index.css');

ET Book
@import url('https://opentype.netlify.com/et-book/index.css');

Sansation
@import url('https://opentype.netlify.com/sansation/index.css');
